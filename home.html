1. What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for HTML and XML documents. It represents the structure of a document as a tree of objects, allowing developers to access and manipulate the content of a webpage using programming languages like JavaScript.

2. How do you select an element from the DOM?

I can select an element from the DOM using methods like document.getElementById(), document.querySelector(), or document.querySelectorAll().

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is an event-handling pattern that allows you to handle events at a higher level in the DOM tree, rather than attaching event listeners to individual elements. This is useful because it reduces the number of event listeners needed, making your code more efficient and easier to manage.

4. How do you manipulate an element's attributes and styles using the DOM?

I can manipulate an element's attributes and styles using the DOM by accessing the element using methods like document.getElementById() or document.querySelector(), and then using properties like setAttribute() to change attributes, and style to change styles. For example, element.style.color = 'red'; changes the text color of the element to red.
